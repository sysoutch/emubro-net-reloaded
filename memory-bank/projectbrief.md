# Project Brief: emuBro - Modern Retro Launcher

## Overview
emuBro is a modern, next-generation retro game launcher and frontend designed to simplify the emulation experience. It aims to provide a seamless home for retro games, handling the technical complexities of emulator association and platform detection so users can focus on playing.

**Current Version:** v2.0 Beta

## Core Goals
- **Simplify Emulation:** Automate platform detection and emulator association.
- **Modern UI/UX:** specific aesthetic with glassmorphism, animated backgrounds, and responsive design.
- **Centralized Hub:** A single place for games, emulators, tools, and community features (netplay, wiki).
- **User-Centric:** "You do the playing, we handle the technical stuff."

## Key Features (Current Prototype)
- **Single Page Application (SPA) feel:** Seamless navigation between views without page reloads.
- **Cloud Saves:** Support for cloud saving functionality (as advertised in v2.0 Beta).
- **Theme Support:** Dark/Light mode toggle.
- **Search:** Quick find functionality (Cmd/Ctrl + K).
- **Dynamic Content:** Javascript-driven lists for emulators and games.
- **Responsive Design:** Mobile-friendly layout.
